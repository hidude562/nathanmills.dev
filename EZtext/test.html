EZ text classifier

Classify text as being literally anything with some degree of accuracy. Input some datapoints and their outputs and you are good. I recomend have atleast a few datapoints for each catagory

Obviously, you can use a generic text generator like gpt-3 or even gpt-2 to do this job for you but that requires a lot of computation power to run. This can be used with any text classifier in theory

To use, you can edit the data by clicking on the table entries, you can add a new row by clicking the button. Once you think you have enough data click "Generate clssifier". Now you are able to use the new algorithm at the bottom of the page
