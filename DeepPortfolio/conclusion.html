Conclusion

In conclusion, deep learning is a powerful tool that can be used to solve complex problems. However, if deep learning continues to be an unsupervised technology, it will continue to become more abused as the technology advances. Although with proper supervision, deep learning has the potential to be a valuable resource for the future.
